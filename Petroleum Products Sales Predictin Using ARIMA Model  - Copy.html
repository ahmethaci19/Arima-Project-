From the above graph, we see that rolling mean itself has a constant line with time, also rolling standard deviation is fairly constant with time. For our time series to be stationary, we need to ensure that both the rolling statistics ie: mean & std. dev. remain time invariant or constant with time. Thus the curves for both of is parallel to the x-axis, which in our case is stationary.

print('Results of Dickey Fuller Test:')

dfoutput = pd.Series(dftest[0:4], index=['Test Statistic','p-value','#Lags Used','Number of Observations Used'])

    dfoutput['Critical Value (%s)'%key] = value

For a Time series to be stationary, its ADCF test should have:

1- p-value to be low (according to the null hypothesis)

2- The critical values at 1%,5%,10% confidence intervals should be as close as possible to the Test Statistics

From the above ADCF test result, we see that p-value is very low. Also critical values are close to the Test Statistics. Hence, we can safely say that our Time Series at the moment is stationary
